KC-CNN-LSTM 瑞雷波频散反演部分代码发布包

一、公开内容
1. 野外实测数据输入的基阶和多模态训练代码；
2. 野外实测数据的基阶和多模态预测以及画图代码；
3. 公开的预训练模型代码以及主程序代码，以及示例（即输入频散数据即可预测）；
4. 核心模块可执行P-code（包含马尔科夫链生成数据、LOSS损失、自定义损失函数与MSE损失函数程序、QPSO寻优程序、基阶与多阶理论频散曲线快速矢量法）
5. 输入数据多模态与基阶模态对比训练代码
6. 传统方法LM、WOA代码以及以及示例画图代码
7. 速度模型-频散曲线画图代码
二、使用说明
将本目录及其子目录加入MATLAB路径，放入预训练模型后运行examples目录中的示例。
